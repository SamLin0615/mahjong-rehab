# Mahjong Rehab

A React-based Mahjong training app focused on:
- Tenpai & wait recognition
- Shanten calculation
- Discard efficiency (ukeire) practice

## Features
- Win Training
- Efficiency Training
- Accurate shanten & ukeire analysis
- Configurable tile count and suit count

## Tech Stack
- React 18
- Create React App
- Docker (optional)

## Development

### Without Docker
```bash
npm install
npm start
```

### With Docker
```
docker-compose up --build
```
